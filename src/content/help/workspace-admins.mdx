## Admin vs member permissions

| Action | Member | Admin |
|---|---|---|
| Use tools and Copilot | ✓ | ✓ |
| Generate and save documents | ✓ | ✓ |
| View shared library | ✓ | ✓ |
| View employee profiles | ✓ (limited) | ✓ (full) |
| Edit employee profiles | — | ✓ |
| Invite / remove members | — | ✓ |
| Manage workspace settings | — | ✓ |
| Approve leave / expenses | ✓ (if set as manager) | ✓ |
| Configure approval workflows | — | ✓ |
| View billing information | — | ✓ |
| Cancel or upgrade plan | Owner only | — |

## Promoting a member to admin

1. Go to **Settings → Members**.
2. Click the **···** menu next to the member's name.
3. Select **Make admin**.

The change takes effect immediately. The member receives a notification.

## Removing admin access

1. Go to **Settings → Members**.
2. Click the **···** menu next to the admin's name.
3. Select **Change to member**.

If the person is the workspace **owner**, you cannot remove their admin access without first transferring ownership.

## Transferring workspace ownership

The workspace owner is the person who created the workspace and holds billing responsibility. To transfer ownership:

1. Go to **Settings → Workspace**.
2. Click **Transfer ownership**.
3. Select the new owner from your current admin list.
4. Confirm the transfer.

The new owner becomes responsible for billing. The previous owner remains an admin unless you manually downgrade them.

## Number of admins

There is no limit to the number of admins you can have in a workspace.

## A note on billing

Only the workspace **owner** can cancel the subscription or change the payment method. Admins can view billing information but cannot make billing changes.
