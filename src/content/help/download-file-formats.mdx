## Available formats

Atlas HR can export generated documents in three formats:

| Format | Extension | Best for |
|---|---|---|
| Microsoft Word | `.docx` | Further editing, sending to lawyers or stakeholders |
| PDF | `.pdf` | Signing, archiving, sending to employees |
| Plain text | `.txt` | Pasting into your own systems or emails |

## How to download

1. Generate or open a saved document.
2. Click the **Download** button (↓) in the document toolbar.
3. Select your preferred format.
4. The file downloads immediately — no email required.

## Format notes

**Word (.docx)**: Preserves heading structure, bold/italic formatting, and tables. Ideal if you need to edit the document in Word or Google Docs before using it.

**PDF**: Fixed layout. Best for documents that should not be further edited — signed offers, completed performance reviews, termination letters. PDFs are not suitable for forms employees need to fill in.

**Plain text (.txt)**: All formatting stripped. Useful for pasting into HR systems, ATS platforms, or emails where rich formatting causes issues.

## Downloading from your library

You can download any previously saved document at any time from your **Library**. Downloading a saved document does not use an additional generation credit.

## Batch downloads

Batch downloading multiple documents at once is available on the Team plan. Select multiple documents in your library and click **Download selected**.
