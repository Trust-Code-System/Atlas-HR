## Yes — always review before using

Atlas HR generates documents as editable drafts. Every generated document should be reviewed by a qualified HR professional or legal adviser before being used in an employment context.

## Editing in the browser

After generation, the document appears in a preview panel. You can:

- **Edit the text directly** — click any section in the preview to enter edit mode.
- **Regenerate specific sections** — use the **Regenerate section** option to re-run a specific paragraph with different instructions.
- **Regenerate the whole document** — click **Regenerate** with updated inputs to produce a fresh version.

Changes made in the browser editor are saved automatically if you save the document to your library.

## Editing a saved document

To edit a document you've already saved:

1. Go to **Library** in the sidebar.
2. Find the document and click on it.
3. Click **Edit** to re-enter the editor.

Edits to saved documents are versioned — you can see previous versions by clicking **History**.

## Downloading for further editing

Download the document as a Word file (`.docx`) or PDF and edit it in your preferred application. Changes made outside Atlas HR are not synced back to your library.

## Regenerating vs editing

| | Regenerate | Edit |
|---|---|---|
| Uses a generation credit | Yes | No |
| Changes inputs | Yes | No |
| AI rewrites content | Yes | No |
| You control the changes | No | Yes |

For small corrections (names, dates, clauses), use **Edit**. For substantive content changes, use **Regenerate** with updated inputs.
