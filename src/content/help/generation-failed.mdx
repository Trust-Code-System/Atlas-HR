## Common causes of generation failures

**1. Generation limit reached**
If you're on the free plan and have used all 3 monthly generations, the Generate button is disabled — this is not an error, it's a limit. Upgrade to Pro or wait until the 1st of next month.

**2. Temporary server issue**
Atlas HR's AI generation can occasionally time out during high-traffic periods. Wait 30 seconds and try again.

**3. Input too long**
If you pasted a very long piece of text into an input field, the generation may fail because the total request exceeds the AI's context limit. Try shortening or summarising the input.

**4. Network issue**
A dropped internet connection mid-generation causes a failure. Check your connection and try again.

## What to do when generation fails

1. **Wait and retry** — Click **Generate** again after a short wait.
2. **Refresh the page** — If the page is in a broken state, refresh and re-enter your inputs.
3. **Try a different browser** — Occasionally, browser extensions (ad blockers, security tools) interfere with the streaming response. Try in incognito mode or a different browser.
4. **Simplify your inputs** — If the failure is consistent, try shorter input text or fewer details.

## Checking system status

If you're consistently seeing failures, check our [status page](/status) for any ongoing incidents. If there's a reported incident, follow it for updates.

## Was I charged a generation credit?

If the generation failed before producing any output, the generation credit is **not** consumed. A generation credit is only counted when the AI successfully starts producing output. If you believe you were incorrectly charged, [contact support](/help/contact).

## Persistent failures on a specific tool

If one tool consistently fails while others work, [contact support](/help/contact) and include:

- Which tool you were using
- What inputs you entered (anonymise sensitive data)
- The error message you saw (screenshot if possible)
- Your browser and operating system
